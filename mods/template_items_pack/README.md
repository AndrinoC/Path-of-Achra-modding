# Template Class Pack

Use this template when you want a custom class to start with custom gear.

What it includes:

- a custom class: `TemplateSentinel`
- a custom skill: `TemplateBattleTrance`
- a custom weapon: `TemplatePracticeBlade`
- a custom chest item: `TemplateTravelCloak`

How it works:

1. enable the mod in the Mods menu
2. restart the game
3. pick `Template Sentinel`
4. start a run and confirm the class spawns with its custom weapon and cloak

Rename first:

- `TemplateSentinel`
- `TemplateBattleTrance`
- `TemplatePracticeBlade`
- `TemplateTravelCloak`

This template is disabled by default through `enabled_by_default: false`.
