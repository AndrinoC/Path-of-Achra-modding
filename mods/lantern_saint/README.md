# Saint of the Last Lamp

This showcase mod adds:

- a custom god: `SaintOfLastLamp`
- a custom skill: `BondOfLastLamp`
- a custom ally: `CoalMoth`
- a custom buff: `LampblackWard`

Theme:

- a soot-lit astral prayer style built around warding, summoned lamp-spirits, and nearest-target punishment

Why it is useful:

- shows custom gods
- shows custom skills with runtime `trait_effects`
- shows a custom ally and buff working from the same prayer trigger
- uses only mod-local art for the showcased content
