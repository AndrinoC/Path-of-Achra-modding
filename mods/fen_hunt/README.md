# Fen Hunt

This showcase mod adds:

- a normal-world enemy: `FenRaider`
- a void-tier enemy: `VoidPilgrim`

Theme:

- marsh stalkers in the early lands and prayer-scorched wanderers in the void

Why it is useful:

- shows custom enemies merged into the normal spawn pools
- shows a tier-1 enemy and a void-tier enemy side by side
- shows nemesis tab visibility using real spawn-tier rules
- uses only mod-local enemy art
