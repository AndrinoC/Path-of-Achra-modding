# Causeway Warden

This showcase mod adds:

- a custom race: `Mireborn`
- a custom class: `CausewayWarden`
- two custom skills: `Fenblood` and `CausewayDiscipline`
- a custom weapon: `CausewayBlade`
- a custom armor item: `DredgerCloak`

Theme:

- marsh-born road wardens who endure long waits, heal slowly in the fen-mist, and answer threats with patient blunt-force retaliation

Why it is useful:

- shows custom race and class entries
- shows starting class gear from modded weapon and armor data
- shows custom race and class start traits built from the merged skill table
- uses only mod-local art for every showcased entry
